# 지능형 주차 관제 시스템 (Smart Parking System)

## 프로젝트 개요
라즈베리파이 5와 Google Coral TPU를 활용한 엣지 AI 주차 관제 시스템입니다.

## 현재 진행 상황 (v0.1)
- [x] YOLOv8 Custom Dataset 학습 완료 (mAP 0.9+)
- [x] PC 환경 시뮬레이션 코드 (`pc_simulation.py`) 구현 완료
- [ ] Raspberry Pi 5 하드웨어 통합 (부품 배송 대기 중)

## 실행 방법
1. `pip install -r requirements.txt`
2. `python main.py`
